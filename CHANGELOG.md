# Changelog

## [1.0.0] - 2024-12-04

### 핵심 기능
- UTM 파라미터 입력 (Base URL, Source, Medium, Campaign, Term, Content)
- 실시간 URL 생성 및 미리보기
- 다중 행 관리 (추가/삭제/일괄 삭제)
- 행별 URL 복사 및 새 탭에서 열기
- localStorage 자동 저장 (새로고침 시 데이터 유지)

### 저장 및 관리
- 선택 항목 저장 (Saved Tab)
- 저장된 URL 테이블 뷰
- 코멘트 인라인 편집
- 행 선택 및 일괄 삭제

### 키보드 인터랙션
- Notion 스타일 셀/행 모드 전환 (ESC)
- 방향키 네비게이션, Enter로 아래 이동
- 셀/행 범위 선택 (Shift+방향키)
- 행 복사/붙여넣기 (Cmd/Ctrl+C/V)
- 단축키: Undo/Redo (Cmd/Ctrl+Z), 저장 (Cmd/Ctrl+S)

### UI/UX
- Apple liquid glass (glassmorphism) 디자인
- 다크 모노톤 테마 (검정/회색)
- 반응형 테이블 레이아웃
- 토스트 알림 시스템
- 키보드 단축키 안내

### 기타
- UTM 파라미터 가이드 섹션
- SEO 메타/OG 태그
- 파비콘 및 OG 이미지
- Google Analytics 추적
